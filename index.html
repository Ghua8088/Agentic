Quick Start

Getting a Gemini API Key

To use Google's powerful cloud models:

  1. Go to the Google AI Studio.
  2. Sign in and click "Create API key".
  3. In Agentic, go to Settings -> System -> Google Gemini API and paste your key.

Installing Ollama (Local Models)

  1. Download from ollama.com.
  2. Run the installer and ensure the icon appears in your tray.
  3. Pull models via terminal (ollama run llama3) or in Settings -> System.

App Customization

Settings & Notifications

Custom Plugins

How to add Plugins

Agentic is highly extensible. You can add your own Python tools by dropping them into the plugins folder.

Plugin Directory:
%APPDATA%/Agentic/plugins (Windows)
~/Library/Application Support/Agentic/plugins (macOS)
Stricter Gemini Integration

To support stricter tool-calling rules (especially for Gemini), use the @register_tool decorator and Pydantic models for input validation.

# plugin_file.py
from pydantic import BaseModel, Field
from pytron.tools import register_tool

class MyInput(BaseModel):
    topic: str = Field(..., description="Topic to analyze")

@register_tool(description="Analyzes a specific topic using local tools")
def analyze_topic(data: MyInput):
    return {"status": "Success", "data": f"Analyzed {data.topic}"}
Security & Responsibility

Virus Warning: Adding custom scripts involves executing code on your machine. Never add plugins from sources you don't trust, as they could contain malicious code or viruses.

Disclaimer: We are NOT responsible for any system damage, data loss, or security breaches caused by third-party plugins. Users install and run custom code at their own risk.

Frequently Asked Questions

Is my data private?

Yes. Agentic is local-first. All vector memory, file scans, and local model interactions never leave your machine.

Where is the source code?

While Agentic is a closed-source flagship binary, the magic that powers it isn't a secret. It is built entirely on the Pytron-Kit—the same engine you can use to build your own AI apps.

Is it free?

The standard version of Agentic is free to use. We believe in providing elite tools for everyone, powered by the open-source Pytron ecosystem.

Can I build this myself?

Yes. If you want to replicate the bridge between Python logic and beautiful React UIs seen here, our core engine Pytron-Kit is fully documented and open for developers.
